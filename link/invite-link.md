# 邀請ESTER BOT

## 如何邀請機器人

點選邀請連結(目前尚不開放)，選擇要加入的伺服器與賦予的權限，按下「授權」即可讓機器人加入伺服器。

{% hint style="info" %}
可依據需求賦予不同權限，但權限未賦予將可能導致部分功能無法使用。
{% endhint %}
