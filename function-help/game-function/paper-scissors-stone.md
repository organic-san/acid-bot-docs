# /paper-scissors-stone 猜拳

## 指令一覽

### /paper-scissors-stone

和機器人猜一次拳。剪刀會贏布，布會贏石頭，石頭會贏剪刀。

#### 填入選項

* gesture:要出的拳，可以從剪刀、石頭、布中選擇。
