# /record 訊息回顧

## 指令一覽

### /record

找回過往的某一則訊息的全文。

#### 填入選項

* message-id:要回顧的訊息的訊息ID。訊息ID的說明請看[這裡](../basic-illustrate.md#yong-hu-id-bin-dao-id-xun-xi-id)。
* (選填)channel:有要回顧的訊息的頻道。如果沒有填寫，那麼會在發送指令的頻道尋找訊息。
