# /timer 計時器

## 指令一覽

### /timer

產生一個計時器，並在指定時間後發送通知。

#### 填入選項

* (選填)hour:要計時的小時數。
* (選填)min:要計時的分鐘數。
* (選填)sec:要計時的秒數。以上時間參數請至少填入一個。
* (選填)message:要附加在計時器上的訊息。
